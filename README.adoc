== About

Minimalistic {tic-tac-toe} implementation in https://www.rust-lang.org[Rust].

== Usage

To launch {tic-tac-toe} with
https://doc.rust-lang.org/stable/cargo/getting-started/installation.html[
`cargo`], run:

[,bash]
----
cargo run
----
